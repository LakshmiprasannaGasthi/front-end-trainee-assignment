# Dynamic Dashboard

## How to run locally

### 1. Clone Repo / Download Zip

```bash
cd dashboard-app
npm install
```

### 2. Run the project

```bash
npm run dev    # if using Vite
```

or 

```bash
npm start      # if using Create React App
```

### 3. Project Opens at

```
http://localhost:3000/ (CRA) or http://localhost:5173/ (Vite)
```
